# Django Personal Blog Project

A modern, fully functional personal blog built with Python and the Django Web Framework. 

## Description

This project focuses on clean architecture, user-friendly interface, and responsive design using Bootstrap 5.
It is created as a portfolio project to showcase Python and Django development skills.

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript 
- SQLite

##  Features 

- Authenticated users can Create, Read, Update, and Delete (CRUD) blog posts.
- Commenting system allowing to users to engage in discussions and reply to specific comments.
- Dynamic image uploading for each post with automated handling of media files.
- Strict permission checks (with Mixins) ensuring only authors can edit or delete their own content.
- Responsive "Hero" section for the landing page.
- Clean card-based layout with hover effects.
- Pure HTML structure with all styling extracted to a dedicated external CSS file.
